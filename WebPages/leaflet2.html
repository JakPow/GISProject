Dentists in Wales (Tile Layers, Circle Markers, Polygon, Event Listners, GeoJSON, Custom styles )

This map displays locations of Dentists in Wales as CircleMarkers using GeoJSON

A TileLayer is used for the base map, with a content layer overlaying it for areas in Wales.

When an individual area is clicked, a function is called using Even Listeners to overlay the area in a new custom style, to show it has been clicked.
