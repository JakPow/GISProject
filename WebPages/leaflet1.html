Leaflet 1


Markers and Layers (Tile layers, markers)

This map uses two leaflet tile layers. One to display a base map, and one to display train lines in the UK.

Two markers are also displayed, showing the locations of football stadiums.

USW Locations (Tile layers, cirlces, polygons, polylines, custom markers, event handlers, animated setView)

This map displays the locations of all USW campus locations. Each campus is represented by a marker that uses custom icons.

When an icon is clicked, an onClick() handler is used to zoom in on the marker using an animation which will also open a popup which has campus information on.

If the icon is clicked again, the map will zoom back out and close the popup automatically. This is done using IF statements for each location.

A Polygon is used to show the area between all the campuses, which also has an onClick() feature that displays a popup showing this.

A circle is used to show the area between the two campuses in Pontypridd, which also has an onClick() feature that displays a popup showing this.

Polylines are used to connect all three locations cross section.

Clicking on any position on the entire map will also display a popup showing the location that you have clicked on the map.
