Google Maps 1


Basic slippy map

This map is just a basic slippy map, proving the connection to the GoogleMaps API using a developer API key.

Football clubs (custom markers, events, infoWindows)

This map shows locations of football clubs in Wales.

All markers use custom icons sourced from each clubs wikipedia page.

Hovering over the marker displays the name of the stadium.

An onClick event displays an InfoWindow with a link to the club website that opens in a new tab

The onClick also pans to the relevent marker.

Castles in Wales (custom markers, animation)

This map displays the locations of castles in Wales using a for loop iterating an array

The castle icons are sourced from flaticon.com

There is a drop animation when the map renders.

Hovering over each icon displays the name of the castle.

Airport paths (polylines, events, infoWindows)

This map displays polylines representing flights paths out of Cardiff airport.

All polylines have unique styling applied to them

There is a mouseOver event that displays an infoWindow, and a mouseOut event which closes it.

The infoWindow display relevent information about the flight, as well as the point of the coordinates that you are hovered over (dynamic)

Windfarms locations (polygones)

This map displays two polygones that show the areas of windfarms in Wales

Both polygones have unique styling applied to them
