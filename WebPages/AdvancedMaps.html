Advanced Maps


Google - Advanced map 1 (Clustering, Event Handler/ Listener, Custom Markers, InfoWindows)

This map shows locations of Hospitals in England using Clustering.

Clicking on the clustered circles will zoom in and show refind results, you can also manually zoom.

When showing one location of a hospital, a custom icon will be shown as the marker, sourced from flaticon.com

An onClick event handler is used whwenyou click on a marker, which displays an infoWindow which displays the name of the hospital.

The hospital data is sourced from gov.co.uk

Google - Advanced map 2 (Custom controls, Custom Map styling, KML data, Polylines, Event Listeners, InfoWindows)

This map shows various routes to travel from Cardiff to London. The map uses custom styling, as well as custom built control buttons to display the routes.

The buttons uses Event Listeners, so when clicked, the route will appear for the selected button.

EAch route consists of KML data that displays a polyline. When the polyine is clicked, an Info window will appear showing the distance and time the route takes.

The route data is sourced from taking the route links from Google Maps, and then converting the link on gpsvisualizer.com into a GPX file, and then into KML data.

Bing - Advanced map 3 (Clustering, Polygons, Custom controls, Event Handlers, InfoWindows, setView)

This map visually shows the populations of all the capital cities in the United Kingdom by using clustering. The cities with higher populations will have more clusters.

When clikcing one of the custom control buttons, the map will pan to the chosen city and zoom in on it using the setView() function.

Each City is surrounded by a polygon to show the area.

When a polygon is clicked, an Event Handler is used to display an infoWindow with the exact population of the city.
