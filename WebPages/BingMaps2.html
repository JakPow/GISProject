Basic Slippy map

This map is just a basic slippy map, proving the connection to the Microsoft Bing maps API using a developer API key.

Stadiums (GeoJSON Markers, custom style)

This Bing map uses GeoJson to display markers that display two stadiums in Wales.

Custom styling is used rather than a default Bing map style.

Custom Bing Map (Custom markers, events, polylines, polygons, event handlers, info boxes)

This is a custom bing map using all the techniques learned.

Customer icons are used for the pins (markers to display three USW campus locations.)

A red dashed polyline is used to connect all three.

A seperate polygon using the same coordinates as the polyline is used to show the land inbetween them all.

An event handler is used to display an infobox with relevent information when the markers are clicked.

The USW icons are sourced from flaticon.com
